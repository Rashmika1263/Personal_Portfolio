Technologies Used

  • The project aims to create a smart wall in the classroom by combining a cane, smartphone camera, and basic projection system.
  • This technology allows any classroom wall to be transformed into an interactive surface.
  • The concept revolves around utilizing a smartphone as a webcam and employing a straightforward Python program on a computer to enable this functionality.

Several challenges were overcome during the development of this project:

  1. Integration of Hardware Components:

    Bringing together a cane, smartphone camera, and projection system required seamless integration of different hardware components. Ensuring compatibility and synchronization between these elements posed a challenge.

  2. Accuracy and Reliability:

    Developing a Python program capable of accurately tracking the movement of a specified colored bead on the wall presented difficulties. Achieving reliable tracking across different lighting conditions and classroom environments required careful consideration.

  3. Cost-Effectiveness:

    Ensuring that the solution remains cost-effective was crucial. The project aimed to utilize existing hardware resources like smartphones and low-end PCs, which demanded optimization to provide a feasible and affordable solution for classrooms.

The project's solution involves the following key elements:

  1. Smartphone Camera Integration:

    By using a smartphone's camera as a webcam, the team harnessed the device's capabilities for capturing visual information.

  2. iVcam App Utilization:

    The iVcam mobile app was identified as a suitable tool for transforming a smartphone into a webcam for any PC over Wi-Fi. This established the foundation for remote camera functionality.

  3. Python Tracking Program:

    To address the need for interactive tracking, a Python program was developed and integrated into the teacher's PC. This program is designed to track the movement of a specific colored bead on the smart wall.

  4. Low-End PC Compatibility:

    The Python program's simplicity and low computational requirements ensured it could run on low-end PCs without performance issues. This strategic decision enhanced the project's cost-effectiveness.

  1. An Android mobile application "iVcam Webcam".
  2. A Python Program which Tracks the bead from input of the Web Cam. (Using OpenCV Module)
